# Project Overview

This repository serves as my final portfolio for the CSE0001 course. Throughout this semester, I have taken on the role of a Student Digital Ambassador. My goal was to build a collection of resources that help my peers handle the challenges of the digital world—from setting up a professional online presence to understanding the legal and technical sides of cybersecurity.

This project is a direct reflection of my learning journey and my commitment to promoting safe, ethical, and effective technology use within our campus community.

**Module Summaries**

The repository is organized into five main folders, each corresponding to a specific task from the course modules:

# Task 1: Digital Literacy Awareness Infographic

Folder: task-1-presentation/

Focus: I designed a visual guide to introduce students to the core concepts of digital literacy. It breaks down safe internet habits and professional etiquette into simple, scannable points.

# Task 2: Student Digital Portfolio

Folder: task-2-portfolio/

Focus: This section documents my professional footprint. It includes proof of my profiles on GitHub, LinkedIn, and Kaggle, which I have optimized to showcase my education and career goals in the cybersecurity field.

# Task 3: Coding and Collaboration Platforms

Folder: task-3-platforms/

Focus: Here you will find evidence of my progress on HackerRank as I build my Python skills. It also includes a link to a collaborative quiz I helped create using Google Forms to test peer awareness of digital safety.

Live Quiz Link: https://forms.gle/QUsS4FD95GMkzNLg8

3 Task 4: Professional Email and Etiquette

Folder: task-4-email-etiquette/

Focus: This folder contains formal email drafts I wrote for academic and professional scenarios, such as requesting a deadline extension and inquiring about an internship. It also includes a checklist for responsible social media behavior.

3 Task 5: Cybercrime Awareness and Prevention

Folder: task-5-cybercrime/

Focus: As a cybersecurity student, I focused this task on a case study involving identity theft and financial fraud. I have also included an 8-point prevention guide that highlights how to use the National Cyber Crime Portal.
